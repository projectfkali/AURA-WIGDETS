## ✨ Özellikler

### 🎨 Glassmorphism 2.0 (Cam Efekti)
Aura Widgets'in kalbinde kusursuz bir cam efekti yatar. Her widget, masaüstü arkaplanınızı bulanıklaştıran *(backdrop-blur)*, sınırlarını belirleyen ve harika hissettiren bir tasarıma sahiptir.
- **Opaklık ve Bulanıklık Kontrolü:** İster tam şeffaf, ister buzlu cam.
- **Köşe Yuvarlama:** Keskin kenarlardan, tam oval pill tasarımlara kadar.

<br>

### 🧲 Manyetik Izgara (Snap-to-Grid) Sistemi
Dağınık masaüstü devri bitti. Widget'ları sürüklerken fiziksel bir yay gibi esneyen (Framer Motion) ve bıraktığınızda 20 piksellik görünmez ızgaralara mıknatıs gibi yapışan kusursuz bir hizalama motoru.

<br>

### 🧠 Akıllı (Smart) Widget: Kodsuz Üretim
Sadece bir URL yapıştırarak internetteki tüm verileri masaüstünüze çekin:
- Döviz/Kripto kurları
- GitHub yıldız sayıları
- Hava durumu verileri
**Kod yazmadan saniyeler içinde kendi API takipçinize dönüştürün.**

<br>

### 🎧 Gerçek Medya Kontrolcüsü
Masaüstünüzde duran "Medya Oynatıcı" modülü süs değildir! Sanal klavye devresini (Virtual Media Keys) kullanarak Spotify, YouTube veya arka planda çalan tüm medyayı doğrudan yönetir. Müzik çalmıyorsa, kapağa tıklayıp varsayılan müzik uygulamasını başlatabilirsiniz.

<br>

### 🌍 Küresel Temalar ve İçe/Dışa Aktarma
Kurduğunuz harika masaüstü düzenini tek tıkla `.json` olarak yedekleyin. İstediğiniz zaman geri yükleyin veya tek tıkla **Cyberpunk**, **Minimalist** ya da **Dark Mode** temalarına geçiş yapın.

## 🚀 Yerleşik Widget Galerisi
Aura Widgets, ilk kurulumda harika yerleşik widget'larla birlikte gelir:
1. **Medya Oynatıcı:** Sistem sesini yönetin.
2. **Ağ ve Batarya:** Şarjınızı ve Wi-Fi kalitenizi izleyin.
3. **Sistem Monitörü (SysMon):** RAM ve CPU takibi.
4. **Kripto Takipçisi:** Anlık Coin fiyatları.
5. **Saat ve Tarih:** Modern dijital saat.
6. **Yapışkan Not (Sticky Notes):** Masaüstü not defteri.
7. **Hava Durumu:** API tabanlı sıcaklık.
8. **Pomodoro:** Odaklanma sayacı.
9. **Özel Kod (Custom):** HTML/CSS/JS gömebildiğiniz iframe kutusu.

## 🛠️ Kurulum

Aura Widgets'ı bilgisayarınızda çalıştırmak için **Node.js** yüklü olmalıdır.

```bash
# 1. Repoyu bilgisayarınıza klonlayın
git clone https://github.com/projectfkali/AURA-WIGDETS.git

# 2. Klasörün içine girin
cd AURA-WIGDETS

# 3. Gerekli kütüphaneleri yükleyin
npm install

# 4. Uygulamayı başlatın!
npm start
```

Uygulama açıldıktan sonra, sistem tepsisindeki (sağ alt köşe, saat yanı) şimşek ikonuna sağ tıklayarak **Dashboard (Ayarlar)** menüsünü açabilirsiniz.
