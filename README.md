# Aura Widget Engine v2.0 (Alpha)

Aura, React, Vite ve Electron teknolojileri ile geliştirilen; masaüstünüzü devasa ve güçlü modüllerle donatmanızı sağlayan bir masaüstü widget motorudur. Apple'ın iOS 26'da kullandığı "Liquid Glass" ve transparan tasarım ilkelerinden ilham alınarak geliştirilmektedir.

## ⚠️ Önemli Uyarı: Geliştirme Aşamasında (WIP)

Aura Widget Engine şu anda **Aktif Geliştirme (Alpha)** aşamasındadır.
- Çok fazla eksik özellik bulunmaktadır.
- Bazı modüllerde hatalar (bug'lar) ve optimizasyon sorunları yaşanabilir.
- Performans sorunları olabilir.
- API limitleri aşıldığında (Örn: Hava Durumu, Kripto, Borsa) widget'lar çalışmayabilir.

Lütfen bu hataları göz önünde bulundurarak test edin.

## Özellikler

- **19 Farklı Widget:** Borsa, GitHub Heatmap, Hava Durumu, Kripto (Top 20), RSS Haberleri, 3D Core, Notlar, Pomodoro ve daha fazlası...
- **Canlı Önizleme:** Widget ayarlarını yaparken anında canlı önizleme panelinde değişiklikleri görebilirsiniz.
- **Şeffaf "Liquid Glass" Tasarım:** Her widget'ın köşelerini, bulanıklık (blur) efektlerini, şeffaflık seviyelerini tamamen ayarlayabilir, isterseniz şeffaf (kapsül) moduna alabilirsiniz.
- **Kişiselleştirme:** Dünyadaki herhangi bir şehrin hava durumunu bulma, farklı haber kaynaklarını seçebilme.
